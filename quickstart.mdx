<Steps>
  <Step title="Create an Account">
    Sign up by entering your name, email, and password, or use a social provider like Google, GitHub, or GitLab.
  </Step>

  <Step title="Configure a Code Management Tool">
    <AccordionGroup>
        <Accordion title="Set Up GitHub" icon="github">
            Connect GitHub via OAuth or token:
              - **OAuth**: Click the "via OAuth" button. You'll be redirected to GitHub's authentication page.
                - Ensure you have `owner` permissions for your organization.
                - Grant access to all repositories you want Kody to analyze (you can adjust this later).
              - **Token**: <a href="/config/github_pat">Visit the GitHub Token Setup page for detailed instructions.</a>
        </Accordion>

        <Accordion title="Set Up GitLab" icon="gitlab">
            Connect GitLab via OAuth or token:
              - **OAuth**: Click the "via OAuth" button, and you'll be redirected to GitLab's authentication page.
                - Grant access to all repositories you want Kody to analyze (you can adjust this later).
              - **Token**: <a href="/config/gitlab_pat">Visit the GitLab Token Setup page for detailed instructions.</a>
        </Accordion>
              <Accordion title="Set Up Bitbucket" icon="bitbucket">
            Connect Bitbucket via OAuth or token:
              - **Token**: <a href="/config/bitbucket_pat">Visit the Bitbucket Token Setup page for detailed instructions.</a>
        </Accordion>
    </AccordionGroup>

    <Warning>If you connect via token, Kody will perform the review using your username.</Warning>

  </Step>

<Step title="Select Repositories">
  Choose the repositories you want Kody to access for automated reviews and
  actionable insights.
</Step>
