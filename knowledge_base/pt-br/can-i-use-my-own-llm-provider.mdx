Sim. O Kodus suporta Bring Your Own Key (BYOK), permitindo que você use seu provedor de LLM preferido.

## Como funciona

1. Vá em **Configurações** → **BYOK** (Bring Your Own Key)
2. Insira sua chave de API do seu provedor preferido
3. O Kodus usa sua chave para todas as operações de IA

## Por que usar seu próprio provedor

- **Controle de dados** — as requisições vão diretamente para o seu provedor
- **Gestão de custos** — use sua relação de cobrança existente
- **Escolha de modelo** — selecione o modelo que funciona melhor para sua base de código
- **Conformidade** — atenda aos requisitos internos para uso de ferramentas de IA

## Provedores de LLM self-hosted

Se você hospeda o Kodus por conta própria, também pode usar provedores de LLM self-hosted como:
- Novita
- Groq
- Together AI
- Fireworks AI

Para guias de configuração, consulte o [Cookbook](/cookbook/pt-br/novita).

Para configuração do BYOK, consulte [Bring Your Own Key](/how_to_use/pt-br/byok).
