title: "¿Puedo usar mi propio proveedor de LLM para la revisión de código?"
description: "Trae tu propia clave API y elige tu proveedor de LLM preferido para la revisión de código con IA en Kodus."
---

Sí. Kodus admite Trae Tu Propia Clave (BYOK, por sus siglas en inglés), lo que te permite usar tu proveedor de LLM preferido.

## Cómo funciona

1. Ve a **Configuración** → **BYOK** (Trae Tu Propia Clave)
2. Ingresa tu clave API para tu proveedor preferido
3. Kodus usa tu clave para todas las operaciones de IA

## Por qué usar tu propio proveedor

- **Control de datos** — las solicitudes van directamente a tu proveedor
- **Gestión de costos** — usa tu relación de facturación existente
- **Elección de modelo** — elige el modelo que funciona mejor para tu base de código
- **Cumplimiento** — satisface los requisitos internos para el uso de herramientas de IA

## Proveedores de LLM alojados de forma autónoma

Si alojas Kodus tú mismo, también puedes usar proveedores de LLM alojados de forma autónoma como:
- Novita
- Groq
- Together AI
- Fireworks AI

Para guías de configuración, consulta el [Cookbook](/cookbook/es/novita).

Para la configuración de BYOK, consulta [Trae Tu Propia Clave](/how_to_use/es/byok).
