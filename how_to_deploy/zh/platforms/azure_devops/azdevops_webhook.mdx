---
title: 创建 Webhook
icon: webhook
---

Kody 通常会自动创建 webhook。如果没有发生，请按照以下步骤手动设置：

1. 导航到您的 Azure DevOps 项目。
2. 单击左下角的"项目设置"。
3. 在"常规"部分下，选择"服务钩子"。
4. 单击"+ 创建订阅"按钮。
5. 是时候配置您的 webhook 了：

   - **服务**：选择 **"Web Hooks"**
   - **在此类型的事件上触发**：选择事件类型（见下文）
   - **URL**：输入您想要接收通知的 URL —— 与您在 `.env` 文件中的 `GLOBAL_AZURE_CODE_MANAGEMENT_WEBHOOK` 中设置的相同。
   - **详细信息**：
     - **拉取请求已创建**
     - **拉取请求已更新**
     - **拉取请求已评论**
   - **过滤器**（可选）：如果需要，您可以按仓库或分支进行过滤。
   - **操作**：配置为向给定的 URL 发送 POST 请求。

6. 单击"完成"

就是这样！您已成功设置 webhook，使 Kody 能够在 Azure DevOps 中审查您的拉取请求。

<Note>
  确保您使用的 URL 可访问并接受拉取请求事件的传入 POST 请求。
</Note>
