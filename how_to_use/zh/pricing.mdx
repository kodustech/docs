## 社区版 vs. 团队版 vs. 企业版

Kodus 提供三个版本:

- **社区版:**免费使用和修改。适用于自托管和云部署(BYOK 模式)。

  - 自带 API 密钥(BYOK)
  - 无限代码审查
  - 无限用户
  - Quality Radar 上的无限工单
  - Kody 学习和记忆
  - Kodus MCP
  - 最多 10 条 Kody 规则
  - 最多 3 个插件/MCP(包括默认 Kodus MCP)
  - ❌ 无驾驶舱分析(仅团队计划)
  - 通过 Discord 的社区支持

- **团队版:**具有附加功能的付费版本。仅适用于云部署。

  - 社区版的所有功能,外加:
  - 无限 Kody 规则(无 10 条规则限制)
  - 无限插件/MCP(无 3 个插件限制)
  - 1 键工单自动修复(Quality Radar)
  - Kodus 分析(驾驶舱)
  - 24 小时响应时间的电子邮件支持
  - 自动更新和维护
  - **两种定价模式:**
    - **BYOK 模式:**$10/开发/月 - 自带 API 密钥以获得最大灵活性
    - **托管模式:**$30/开发/月 - 我们为您管理令牌,成本固定

- **企业版:**为企业使用定制的版本。可在本地和云中使用。
  - 团队版的所有功能,外加:
  - Kody 代理的优先队列
  - RBAC + 审计日志和分析
  - 专用实例
  - 自定义 SLA
  - 高级安全功能
  - SSO 和 SAML 集成
  - 专门的支持经理
  - 培训和入职协助
  - 定制选项

## 定价如何工作

我们的定价是按团队中主动创建拉取请求的开发人员计费。
这确保了随着团队发展的灵活性和可扩展性:

- **按活跃开发人员付费:**仅为主动贡献拉取请求的开发人员付费。
- **随时可调整:**随时通过您的账户设置更新活跃开发人员的数量。

## 团队定价:BYOK vs. 托管

选择最适合您团队需求的模式:

### BYOK 模式 - $10/开发/月

自带密钥(BYOK)为您提供最大的灵活性和控制:

- **您管理自己的 API 密钥** 来自任何 LLM 提供商
- **选择最佳模型** 适合您的用例 - 无论是最先进的还是最具成本效益的
- **完全透明** - 您确切地看到您在令牌上花费了多少
- **无令牌限制** - 根据您提供商的限制使用您需要的任意多
- 非常适合想要优化成本或使用尖端模型的团队

### 托管模式 - $30/开发/月

让 Kodus 以最佳性价比为您处理一切:

- **我们管理所有令牌** - 无需设置 API 密钥
- **优化价值** - 我们智能地选择模型并优化使用以适应每开发 $20/月的令牌,确保您以最佳成本获得最佳质量
- **固定成本保证** - 每开发 $30/月总计,无意外
- **自动优化** - 我们平衡质量和成本,类似于 Cursor 的自动模式
- **零配置** - 只需注册并开始审查
- 非常适合想要最大价值而不复杂管理 API 密钥的团队

两种模式都包括所有团队功能 - 唯一的区别是您如何处理 API 密钥和令牌成本。

## 无限拉取请求

您的团队每周可以提交任意多的拉取请求——所有计划都没有 PR 限制。唯一的限制来自您的 Git 提供商的 API 限制和非常大的代码库。我们不允许审查超过 200 个文件的拉取请求。

**如何申请:**

1. 加入我们的 Discord 社区 [discord.gg/TFZBRk9fT6](https://discord.gg/TFZBRk9fT6)
