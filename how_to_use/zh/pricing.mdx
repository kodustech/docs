## 社区版 vs. 团队版 vs. 企业版

Kodus 提供三个版本:

- **社区版:**免费使用和修改。适用于自托管和云部署(BYOK 模式)。

  - 自带 API 密钥(BYOK)
  - 无限代码审查
  - 无限用户
  - Quality Radar 上的无限工单
  - Kody 学习和记忆
  - Kodus MCP
  - 最多 10 条 Kody 规则
  - 最多 3 个插件/MCP(包括默认 Kodus MCP)
  - ❌ 无驾驶舱分析(仅团队计划)
  - 通过 Discord 的社区支持

- **团队版:**具有附加功能的付费版本。仅适用于云部署。

  - 社区版的所有功能,外加:
  - 无限 Kody 规则(无 10 条规则限制)
  - 无限插件/MCP(无 3 个插件限制)
  - 1 键工单自动修复(Quality Radar)
  - Kodus 分析(驾驶舱)
  - 24 小时响应时间的电子邮件支持
  - 自动更新和维护
  - **BYOK 模式:**$10/开发/月 - 自带 API 密钥以获得最大灵活性

- **企业版:**为企业使用定制的版本。可在本地和云中使用。
  - 团队版的所有功能,外加:
  - Kody 代理的优先队列
  - RBAC + 审计日志和分析
  - 专用实例
  - 自定义 SLA
  - 高级安全功能
  - SSO 和 SAML 集成
  - 专门的支持经理
  - 培训和入职协助
  - 定制选项

## 定价如何工作

我们的定价是按团队中主动创建拉取请求的开发人员计费。
这确保了随着团队发展的灵活性和可扩展性:

- **按活跃开发人员付费:**仅为主动贡献拉取请求的开发人员付费。
- **随时可调整:**随时通过您的账户设置更新活跃开发人员的数量。

## 团队定价

团队版采用 BYOK 模式:

### BYOK 模式 - $10/开发/月

自带密钥(BYOK)为您提供最大的灵活性和控制:

- **您管理自己的 API 密钥** 来自任何 LLM 提供商
- **选择最佳模型** 适合您的用例 - 无论是最先进的还是最具成本效益的
- **完全透明** - 您确切地看到您在令牌上花费了多少
- **无令牌限制** - 根据您提供商的限制使用您需要的任意多
- 非常适合想要优化成本或使用尖端模型的团队

## 无限拉取请求

您的团队每周可以提交任意多的拉取请求——所有计划都没有 PR 限制。唯一的限制来自您的 Git 提供商的 API 限制和非常大的代码库。我们不允许审查超过 200 个文件的拉取请求。

## 对非营利组织和开源的支持

我们相信支持有影响力的工作。Kodus 对非营利组织和开源项目完全免费。

**如何申请:**

1. 加入我们的 Discord 社区 [discord.gg/TFZBRk9fT6](https://discord.gg/TFZBRk9fT6)
2. 在社区中分享您的组织名称、网站和工作的简要描述
3. 对于非营利组织:提及非营利状态证明
4. 对于开源:分享您的公共代码库链接
5. 我们将在 3 个工作日内审查您的申请

## 付款和计费

- **由 Stripe 提供支持:**付款使用 Stripe 安全处理,这是一个全球信赖的平台。
- **随时取消:**您可以随时取消您的计划。更改将在您的计费周期(月度或年度)结束时生效。
- **发票和巴西税务发票:**对于巴西客户 🇧🇷,我们为每笔交易提供发票并开具税务发票(税务收据)。
