## Modelos utilizados

A Kodus utiliza modelos de linguagem de ponta para processamento de dados não estruturados e análise avançada. Os modelos atualmente em uso são:

- **GPT-4o e derivados**: Modelo de linguagem avançado desenvolvido pela OpenAI, conhecido por sua capacidade de compreensão e geração de texto de alta qualidade.
- **Gemini 1.5 Pro e derivados**: Modelo multimodal da Google, capaz de processar e gerar texto, imagens e outros tipos de dados.
- **Claude 3.5 Sonnet e derivados**: Modelo de linguagem da Anthropic, projetado com foco em segurança e alinhamento ético.

## Tratamento de dados

### Uso temporário
Todos os dados processados pelos modelos são utilizados apenas para contextualização temporária. Isso significa que:

- Os dados são mantidos apenas durante a sessão de processamento.
- Após a conclusão da tarefa, os dados são imediatamente descartados.

### Não utilização para treinamento
Conforme estabelecido nos contratos com os fornecedores dos modelos:

- Nenhum dado do cliente é utilizado para treinar ou melhorar os modelos.
- A privacidade e confidencialidade das informações são rigorosamente mantidas.

## Medidas de segurança

Para garantir a proteção dos dados dos nossos clientes, implementamos as seguintes medidas:

1. **Criptografia**: Todos os dados são criptografados em trânsito e em repouso, utilizando protocolos de criptografia de ponta.

2. **Isolamento de dados**: Os modelos são projetados para não armazenar ou reter dados após o processamento.

3. **Acesso controlado**: Implementamos rigorosos controles de acesso para garantir que apenas pessoal autorizado possa interagir com os sistemas que processam dados do cliente.

4. **Auditorias regulares**: Realizamos auditorias de segurança periódicas para garantir a conformidade com as melhores práticas do setor.

5. **Atualizações de segurança**: Mantemos todos os sistemas e modelos atualizados com as últimas correções de segurança.

## Compromisso com a privacidade

A Kodus está comprometida em manter os mais altos padrões de privacidade e segurança de dados. Estamos em conformidade com regulamentações de proteção de dados aplicáveis e continuamente aprimoramos nossas práticas para proteger as informações dos nossos clientes.

Para mais informações sobre nossas políticas de privacidade e segurança, entre em contato com nossa equipe de suporte.
